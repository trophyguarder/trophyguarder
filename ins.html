InSight Lander Twitter Bot - Mars Weather

During quarantine, I figured that I might as well pick up a new skill or two. I already have a wealth of experience with MATLAB, but almost none with object-oriented programming. To fix this, I wanted to start working on a series of personal projects using Python. The first project I settled on was a Twitter bot that posts live weather updates from Elysium Planitia on Mars. You can see an example Tweet from the bot below:

Weather at Elysium Planitia for Sol 515 (May 09, 2020) - Data from @nasa InSight Lander pic.twitter.com/4mFfWcfiWL

— InSight Lander - Mars Weather (@insight_weather) May 9, 2020

This bot utilizes the NASA Public InSight Lander API to acquire the weather data. A new tweet is composed every 24 hours with an update of the weather on Mars, provided that the InSight API has released new data. The bot is hosted on Heroku to keep it running at all times. You can find the full code for this project on my GitHub.
